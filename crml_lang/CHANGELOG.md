# crml-lang Changelog

This changelog covers the `crml-lang` package (language/spec).

## 1.2.0 - 2025-12-21

### Added
- CRML 1.2 Pydantic models
- Bundled CRML JSON Schema (`crml-schema.json`)
- Structured validator (`ValidationReport`, `ValidationMessage`)
- YAML load/dump helpers (`CRModel` + module-level helpers)
- OSCAL integration support for importing OSCAL catalogs
- New attack catalogs (CAPEC, Diamond Model, Kill Chain, NIST CSF, Unified Kill Chain)
- Lesson-based example suite (Lessons 01-06)

### Changed
- Enhanced portfolio bundle validation
- Improved attack catalog validation
- Updated control relationships models
